# Codex task: DESIGN EVOLUTION v2 — improve the approved style, don't copy it bluntly (Naadir's direction)

Baseline: the genuine approved packs in `creative/templates/approved-final/` (June/July). That style is brand equity — evolve it, don't replace it.

## Creative-director critique of the current style (what to keep / fix)
**KEEP (the equity):** cinematic packshot-on-pedestal staging, dark botanical depth, golden rim light, real Shopify packshots, exact WAS/NOW maths, the disclaimer footer.
**IMPROVE:**
1. **Density.** Each ad currently carries 9+ elements (ribbon, title, 2 pills, 3 bullets, price block, validity bar, 4 roundels, CTA, store strip, disclaimer). Premium = space. Target: one hero message per ad; cut to ≤6 elements (drop the roundel row OR the bullets — never both present); more negative space around the product.
2. **Audience fit.** The dark+gold grade reads "luxury liquor/festive" more than "premium wellness for women 35–50, high-LSM". Build a TWO-MODE system:
   - **Dark Offer mode** (evolved current): for Mon/Wed/Thu deals. Keep drama; lighten clutter; cooler, fresher botanicals for winter.
   - **Light Editorial mode** (new): airy natural-light premium-wellness grade (think morning kitchen/bathroom light, fresh botanicals, warm neutrals + deep One Life green accents) for education, proof, Vivid story, carousels. This is what her feed aesthetic rewards.
3. **Typography hierarchy.** All-caps condensed everywhere = promo-shouting. Keep condensed caps for offer titles/prices; introduce a calmer editorial weight for education/story lines.
4. **Proof slot.** Add an optional review-stars + short-quote slot (benchmarks: proof beats claims) for when real reviews exist — never fabricated.
5. **Craft details:** real drawn icons (no font glyphs), grounded product shadows, no element ever overlapping the CTA bar, minimum legible font size on mobile.
