# Codex prompt — Cinematic lifestyle banners for Vivid Health

Generate 15 premium lifestyle/banner images for the Vivid Health website.
These are BACKGROUND images — no text, no headlines, no benefit bullets
baked into the image. The website renders HTML text on top. The images
must work as clean editorial photography that looks stunning UNDER text.

## Cinematic direction (applies to ALL images)

- Shot on medium format (Hasselblad / Phase One look)
- f/2.0–2.8 shallow depth of field with creamy bokeh
- Strong directional key light from one side (never flat)
- Warm golden highlights, cool blue-green shadows
- One "imperfect" prop element (herb slightly out of frame, steam rising)
- Physically convincing contact shadows
- NO text, NO logos, NO headlines baked into the image
- 1920×1080px landscape for banners, 1200×1500px portrait for mobile hero
